# Generador de Copy — Dra. Catalina Lefort

## Contexto de Marca
- **Voz:** Profesional, cálida, basada en evidencia, "sin medidas extremas", nunca predicadora
- **Audiencia:** Mujeres chilenas 25–50, frustradas con medicina tradicional
- **Oferta:** Sesión 1:1 de 60 min — $59.91 USD
- **Resultado a prometer:** Energía, claridad, control de salud — no pérdida de peso
- **Idioma:** Español chileno

## Matriz de Consciencia (Siempre Aplicar)

| Etapa | Mentalidad del cliente | Estrategia de mensaje |
|---|---|---|
| Sin conciencia | "Siempre estoy cansada, es normal" | Estadística + patrón de interrupción |
| Consciente del problema | "Mis exámenes me asustaron y nadie explica" | Validar + diferenciar de medicina tradicional |
| Consciente de la solución | "Quiero plant-based pero temo deficiencias" | Superar objeción B12/Fe/proteína con evidencia |
| Consciente del producto | "Ya oí de Dra. Catalina, ¿vale la pena?" | Prueba social + desglosar valor de los 60 min |
| Muy consciente | "Lista para reservar, solo necesito el empujón" | Eliminar fricción + urgencia suave |

## Tipos de Output

### Titulares (cuando se piden)
Producir mínimo 5, organizados por tipo:
1. **Declaración audaz** — afirmación que desafía lo conocido. Ej: "Tu médico nunca tuvo 60 minutos para ti."
2. **Estadística** — dato que impacta. Ej: "El 84% de las mujeres chilenas tiene deficiencia de vitamina D."
3. **Pregunta directa** — activa la mente. Ej: "¿Y si tu cansancio no es normal?"
4. **Promesa de resultado** — beneficio concreto. Ej: "Entiende tus exámenes. Recupera tu energía."
5. **Objeción directa** — nombra el miedo. Ej: "¿Proteína en plantas? Sí, más que suficiente."

### Hooks (cuando se piden)
Producir 3–5 hooks con contexto de cuándo usarlos:
- **Scroll-stop:** estadística impactante o afirmación inesperada
- **Pregunta:** genera respuesta interna inmediata
- **Dato personal:** "Si tienes exámenes recientes y nadie te los explicó..."

### Copy largo (cuando se pide)
Usar framework P.A.S.:
- **P**roblema: nombrar el dolor exacto en las palabras del cliente
- **A**gitación: ¿por qué importa resolver esto? ¿qué pasa si no lo hace?
- **S**olución: Dra. Catalina + la sesión de 60 min como respuesta específica

### Variaciones (cuando se piden)
Tomar el copy existente y producir 3 versiones con cambios de:
- Intensidad del gancho (suave → fuerte)
- Ángulo (mismo dolor, diferente entrada)
- CTA (diferente verbo de acción)

- "Escribe el hook para un anuncio de fatiga" → 3 hooks TOFU energía
- "Variaciones del copy de la sesión" → 3 variaciones BOFU valor/precio
- "Titulares nuevos para el funnel" → 5 titulares orientados a conversión
