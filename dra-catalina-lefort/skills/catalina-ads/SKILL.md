# Generador de Anuncios — Dra. Catalina Lefort

Cuando esta skill se activa, genera un anuncio completo de Meta Ads adaptado exactamente a la marca de Dra. Catalina.

## Contexto de Marca (Siempre Aplica)
- **Quién:** Dra. Catalina Lefort — médica especialista en Medicina del Estilo de Vida, plant-based, Chile
- **Oferta:** Sesión 1:1 de 60 min — $59.91 USD — interpretación de exámenes + plan personalizado
- **Audiencia:** Mujeres chilenas 25–50, nivel medio-alto, frustradas con medicina tradicional
- **Voz:** Profesional, cálida, basada en evidencia, nunca predicadora, "sin medidas extremas"
- **Resultado principal a prometer:** Energía — no pérdida de peso
- **Idioma:** Español chileno

## Proceso Automático

**Paso 1 — Identificar etapa del funnel:**
- ¿Habla de un dolor o problema cotidiano? → **TOFU**
- ¿Aborda una objeción o miedo? → **MOFU**
- ¿Es para cerrar una reserva? → **BOFU**
- Si no es claro, elige TOFU por defecto.

**Paso 2 — Generar el anuncio completo con estos 6 elementos:**

## ANUNCIO: [Nombre del ángulo]
**Etapa:** TOFU / MOFU / BOFU

### 1. Hook
[Frase que detiene el scroll. Usa pregunta directa, estadística impactante, o afirmación inesperada. Max 1 línea.]

### 2. Copy Corto (3 líneas)
Línea 1: [Refleja el dolor exacto del cliente]
Línea 2: [Agita — ¿por qué importa? ¿qué pasa si no lo resuelve?]
Línea 3: [Solución + Dra. Catalina + CTA suave]

### 3. Copy Largo (5–8 líneas)
[Problema → por qué existe → por qué la medicina tradicional falla → Dra. Catalina como solución única → resultado específico → CTA]

### 4. Titular (texto bajo la imagen)
[Max 10 palabras. Beneficio directo. Ej: "Entiende tus exámenes. Recupera tu energía."]

### 5. CTA
Elige uno: "Reserva tu sesión" / "Agenda aquí" / "Quiero mi evaluación" / "Ver disponibilidad"

### 6. Brief de Imagen
- Escena: [descripción concreta]
- Sujeto: [mujer chilena 30–45, expresión específica]
- Colores: [verdes suaves, blanco, tonos tierra, luz natural]
- Elemento de texto: [si aplica]
- Evitar: imágenes frías, hospitalarias, antes/después del cuerpo, propaganda vegana

**Paso 3 — Ofrecer prompt JSON para imagen:**
Al final del anuncio, preguntar: "¿Quieres que genere también el prompt JSON para crear la imagen con IA?"
Si sí → construir el prompt Nano Banana con el ángulo del anuncio.

## Para Variaciones Rápidas
Si el usuario pide "otra variación" o "cambia el hook" → generar SOLO el elemento modificado, no repetir todo.
